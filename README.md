# -FourTerminalNetwork-TwoPortNetwork--CALCULATOR
Calculadora y simulador educativo de cuadripolos / redes de dos puertos. Permite dibujar netlists en vivo, visualizar el esquema eléctrico, resolver matrices Z e Y, derivar automáticamente parámetros h, g y ABCD/Gamma, convertir entre familias de matrices, asociar cuadripolos y exportar resultados técnicos en Markdown, LaTeX, JSON o PDF.
